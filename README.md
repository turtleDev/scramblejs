# scramblejs

> text garble animation

## Demo
<img src="data/demo.gif">

## Usage

```js
/* without jQuery */
scramble.enscramble(<DOMelement>);

/* with jQuery */
$('<selector>').scramble();
```

## Installation
* clone this repo
* run `npm install && npm run build`
* copy the dist/scramble.js to where ever you need it
